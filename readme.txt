=== To Do ===

A full featured plugin for creating and managing a To Do list.

== Description ==

To Do for Wordpress

Decagon Todo is a great Todo Plugin. You can Manage all of your Todo List simply with this complete & right tool with all the options you need!

You can show Todo List in the Frontend placing a simple todo in the URL

== Features ==
* Create todo from the front end of your site.
* Its have the following attribute: Task(text), Status, Created At, Due Date(datetime)
* Add comments on individual todo lists
* Expose a filter (hook) that allows other plugins to alter the Task field before you save it to   the database
* Expose a hook for pre/post phase when storing a Todo
* Mark Task as Completed, Cancle, New etc
* Ajax data table
* Instant search any word within the table/ sorted by Due Date
* Shortcode to use Todo from frontend.

== Shortcodes ==
* todo

== Installation ==
Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
